## Featured Project

### [Recruiting Ops](https://github.com/margaritashots/recruiting-ops)

A prompt and template hub for senior recruiting advisory work. It includes executive hiring briefs, candidate debrief structures, role strategy intake templates, market mapping prompts, outreach guidance, and stakeholder update formats.

The project is designed to help recruiting leaders operate as strategic advisors to executives: lead with recommendations, separate evidence from interpretation, state confidence, and make hiring tradeoffs explicit.
